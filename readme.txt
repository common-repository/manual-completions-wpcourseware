Manual Completions for WP Courseware lets you check completion as well as manually mark courses, units and quizzes as complete. Individuals as well as in bulk.

== Description ==

Manual Completions for WP Courseware provides a very simple interface to check completion as well as manually mark courses, units and quizzes as complete.

You can use it for single completions as well as for **bulk completions** of hundreds of users. In one click, get all the enrolled users in the course.

You can also upload a CSV files with user_id, course_id, module_id, unit_id, quiz_id. To quickly list all the completions, and then process them in bulk in any order you want.

You can also bypass completions blocked by GrassBlade xAPI Companion for xAPI Contents.

**Tracking:**
- If you have an LRS, you can see tracking data, including the user id and name of the admin who marked the unit complete.

**Requirements to use this plugin:**
To use this plugin you need these two plugins:
1. WP Courseware LMS
2. GrassBlade xAPI Companion


**Related Plugins:**
- [Visibility Control for WP Courseware](https://wordpress.org/plugins/visibility-control-for-wpcourseware/)
- [Experience API for WP Courseware by Grassblade](https://wordpress.org/plugins/grassblade-xapi-wp-courseware/)


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/manual-completions-wpcourseware` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to WP ADMIN > GrassBlade > Manual Completions for WP Courseware or (WP ADMIN > WP Courseware > Manual Completions)
